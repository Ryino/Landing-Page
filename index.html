Welcome to DineHub

DineHub is a student-driven project created specifically for GIKI students. It serves as a food e-marketplace designed to solve a long-standing problem faced by GIKI residents: the challenge of easily ordering food.

Download APK

Core Capabilities

  • User Interaction: Users can browse menus, place orders, and provide feedback for improvements.
  • Vendor Management: Vendors can manage orders, update menus, and set pricing effectively.
  • Payment Features: Payments can be processed securely with multiple payment options and tracking features.
  • Security Protocols: Transactions can be secured with SSL certificates and HTTPS protocols.
  • User Experience: User experience can be enhanced with a chatbot for support and engagement.
